# PRD — Dream Farm Life Landing Page 🌾

## Overview

Landing page untuk Dream Farm Life — farming simulation game di Solana. Tujuan: inform, excite, convert visitor jadi player.

**URL:** https://dreamfarmlife.com (atau subdomain)
**Target:** Gamer crypto, casual gamer, Solana community

---

## Objective

1. Showcase game features & vibe (relaxing, fun)
2. Drive wallet connection / waitlist signup
3. Show $DREAM token info & NFT preview
4. Link ke playable demo / app
